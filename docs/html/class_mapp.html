vector< vector< Point2< int > > > Mapp::minPathNPoints ( const vector< Point2< int > > &  vp)

Given a couple of points the function compute the minimum path that connect them avoiding the intersection of OBST and BODA.

The function is based on a Breadth-first search (BFS).

Parameters
[in]p0The source point.
[in]p1The destination point.
Returns
A vector of vector of points along the path (one for each cell of the grid of the map). Each vector is the best path for one connection, given n points there are n-1 connecctions.

Given a couple of points the function compute the minimum path that connect them avoiding the intersection of OBST and BODA.

The function is based on a Breadth-first search (BFS).

Parameters

[in]endPThe destination point.
